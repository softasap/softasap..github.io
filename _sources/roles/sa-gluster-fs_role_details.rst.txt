    <h1>sa-gluster-fs</h1>
    <p><a href="https://travis-ci.org/softasap/sa-gluster-fs"><img alt="Build Status" src="https://travis-ci.org/softasap/sa-gluster-fs.svg?branch=master" /></a></p>
    <p>Installs basic gluster fs server package.</p>
    <p>GlusterFS is a scalable network filesystem. Using common off-the-shelf hardware, you can create large, distributed storage solutions for media streaming, data analysis, and other data- and bandwidth-intensive tasks. GlusterFS is free and open source software.</p>
    <p>Example of usage (all parameters are optional)</p>
    <p>Simple</p>
    <p>roles:
        - {
            role: "sa-gluster-fs"
          }</p>
    <p>Advanced:</p>
    <p>roles:
        - {
            role: "sa-gluster-fs",
            glusterfs_version: "3.7"
          }</p>
