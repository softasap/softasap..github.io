    <p><a href="https://travis-ci.org/softasap/sa-mongo"><img src="https://camo.githubusercontent.com/d5e185cbfc3036bb4129457dce0a4b0928862776/68747470733a2f2f7472617669732d63692e6f72672f736f6674617361702f73612d6d6f6e676f2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/softasap/sa-mongo.svg?branch=master" style="max-width:100%;"></a></p>
    
    <p>Role to install MongoDB 2.4 2.6 or 3.* on ubuntu based box.</p>
    
    <p>Version is controlled by  mongo_version parameter.</p>
    
    <p>mongo_version: "3.4"  #  "2.6" | "3" | "3.2" | "3.4"
    mongo_family: "org" # "org" | "enterprise"</p>
    
    <p>Example:</p>
    
    <p>Simple</p>
    
    <div class="highlight highlight-source-yaml"><pre>
         - <span class="pl-s">{</span>
             <span class="pl-ent">role</span>: <span class="pl-s"><span class="pl-pds">"</span>sa-mongo<span class="pl-pds">"</span></span><span class="pl-s">,</span>
             <span class="pl-ent">mongo_version</span>: <span class="pl-s"><span class="pl-pds">"</span>3.4<span class="pl-pds">"</span></span>
           <span class="pl-s">}</span>
    </pre></div>
    
    <p>Advanced:</p>
    
    <div class="highlight highlight-source-yaml"><pre>
         - <span class="pl-s">{</span>
             <span class="pl-ent">role</span>: <span class="pl-s"><span class="pl-pds">"</span>sa-mongo<span class="pl-pds">"</span></span><span class="pl-s">,</span>

    <p>If you installed the sa-mongo role using the command</p>
    
    <p><code>
       ansible galaxy install softasap.sa-mongo
    </code></p>
    
    <p>the role will be available in the folder library\softasap.mongo.
    Please adjust the path accordingly.</p>
